# TechFluent - Educational Website

**TechFluent** is a modern educational website designed to offer a wide range of tech-related courses, connect learners with expert tutors, and help students begin their journey in tech with confidence.

## 🌐 Live Pages
- Home Page: `Home-page.html`
- About Us: `about.html`
- Courses: `courses.html`
- Contact Us: `contact-us.html`

## 📁 Project Structure
```
├── Icons/                  # Icons and logos used across the site
├── Images/                 # Images for courses, tutors, blogs, etc.
├── Video/                  # Video files for the about section
├── about.css               # Styles for the About page
├── about.html              # About page HTML
├── contact-us.css          # Styles for the Contact Us page
├── contact-us.html         # Contact Us page HTML
├── courses.css             # Styles for the Courses page
├── courses.html            # Courses page HTML
├── general.css             # Global styles and floating symbols animation
├── header-footer.css       # Styles for header and footer components
├── home-page.css           # Styles for the Home page
├── Home-page.html          # Home page HTML
└── README.txt              # Project documentation
```

## 🛠 Technologies Used

- **HTML5**
- **CSS3**
- Google Fonts
- Responsive Design Principles


## 📸 Features

- Responsive layout for mobile/tablet/desktop
- Floating tech symbols for visual appeal
- Courses listed by category
- Tutor profiles with social media
- Contact form and Google Map integration
- FAQ section

## 🚀 How to Run

1. Clone or download this repo
2. Open `Home-page.html` in your browser
3. Make sure your `Images/`, `Icons/`, and `Video/` folders are properly placed

## 🛠 Future Improvements
